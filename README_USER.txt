PKBM GENERATOR - PANDUAN PENGGUNA
=====================================

CARA MENGGUNAKAN APLIKASI:

1. Double-click file "PKBM_Generator.exe"
2. Pilih jenis dokumen yang ingin dibuat (Rapot, SKHUPK, atau Transkrip)
3. Pilih file Excel yang berisi data
4. Upload tanda tangan (JPG/PNG):
   - TTD Kepala Sekolah
   - TTD Wali Kelas (untuk Rapot)
5. Pilih folder output untuk menyimpan hasil PDF
6. Klik tombol "Generate PDF"
7. Tunggu proses selesai
8. File PDF akan tersimpan di folder yang dipilih

CATATAN PENTING:

- Aplikasi ini TIDAK memerlukan instalasi Python
- Pastikan file Excel sesuai format template
- Format tanda tangan: JPG atau PNG
- Ukuran tanda tangan akan otomatis disesuaikan
- Semua file PDF akan dibuat dalam folder terpisah per kelas

TROUBLESHOOTING:

Q: Aplikasi tidak bisa dibuka?
A: Klik kanan > Properties > Unblock (jika ada warning dari Windows)

Q: Antivirus mendeteksi sebagai virus?
A: Ini false positive, whitelist aplikasi di antivirus

Q: Error saat generate PDF?
A: Pastikan format Excel sesuai template dan semua data terisi

Q: Tanda tangan tidak muncul?
A: Pastikan file JPG/PNG valid dan tidak corrupt

KONTAK SUPPORT:
Jika ada masalah, hubungi administrator sistem.
